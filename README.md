# Countdown para Notion

Temporizador simples em HTML, CSS e JavaScript, pensado para ser hospedado no GitHub para depois ser incorporado no Notion via embed.

## Funcionalidades

- seleção manual de minutos e segundos
- botões de tempo rápido
- iniciar, pausar e resetar
- alarme sonoro ao chegar em `0:00`
- solicitação de notificação do navegador
- tema claro/escuro
- layout responsivo e clean

## Como incorporar no Notion

1. Copie a URL https://carolgonzaga.github.io/countdown-notion/.
2. Cole a URL do projeto.
3. Clique em Incorporar.
